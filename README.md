
# Students Result App



## Run Locally

Clone the project

```bash
  git clone https://github.com/Abhishek-Kamble/StudentsApp.git
```

Go to the project directory

```bash
  cd StudentsApp
```

Run Jar File

```bash
  java -jar StudentsApp1.jar
```


## Tech Stack

**Jave 17 & JSON Simple library**
